Functions with multiple independent variables

Functions can have more than one independent variable. 

Functions of one independent variable

The following function is a function of one independent variable:

   

\(\PalAa y=f(\PalAb x)=\PalAb x+3\)
where   \(\PalAb x\) is the independent variable
and   \(\PalAa y\) is the dependent variable.


Functions of multiple independent variables

The following functions are all functions of more than one independent variable:

\(y=f(x,z)\) \(=\) \(5x-4z\)
\(w=g(x,y,z)\) \(=\) \(3xy^2z^3\)
\(h(x_1,x_2,x_3,x_4)\) \(=\) \(4x_1-2x_2-5x_3+x_4\)

Note that the notation used for functions of several variables is similar to the notation used for single variable functions.


The following function is a function of two variables:

   

\(\PalAa y=f(\PalAb x,\PalAb z)=5\PalAb x-4\PalAb z\)
where   \(\PalAb x\) and \(\PalAb z\) are the independent variables
and   \(\PalAa y\) is the dependent variable.


Likewise, the following function is a function of three variables:

   

\(\PalAa w=g(\PalAb x,\PalAb y,\PalAb z)=3\PalAb{xy}^2\PalAb{z}^3\)
where   \(\PalAb x\), \(\PalAb y\) and \(\PalAb z\) are the independent variables
and   \(\PalAa w\) is the dependent variable.


The function \(h\) is similar except there are four independent variables.
