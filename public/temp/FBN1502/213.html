Summary

  • The general equation for an exponential function is \(\PalAa{y=f(x)=a^x}\) where \(a>0\), \(a\neq1\) and \(a\) is a real constant.
     
  • If \(\PalAa{a>1}\) the exponential function is an increasing function of \(x\) and 
    if \(\PalAb{0<a<1}\) the exponential function is a decreasing function of \(x\).
     
  • Exponential functions are used to model compound interest, radioactive decay and various types of growth (populations, bacterial or animal).
     
  • An inverse function is a function that reverses another function:
    If the function applied to an input \(\PalAa x\) gives a result of \(\PalAb y\), then applying the inverse function to \(\PalAb y\) gives the result of \(\PalAa x\).
     
  • The logarithmic function is completely determined by the corresponding exponential function.  They are inverse functions.  If \(\PalAa{f(x)=a^x=y}\) then \(\PalAb{\log_ay=x}\) with y > 0.

Specific cases of logarithmic functions:

\(\log_aa=1\)

This is because \(a^1=a\).

\(\log_a1=0\)

This is because \(a^0=1\).

\(\log_{10}y\)

The base is \(10\) and the logarithm is called
the common logarithm.

\(ln\)

The base is \(e\) and the logarithm is called
the natural logarithm.

\(lg\)

The base is \(2\) and the logarithm is called
the logarithm base \(2\) in computer science.
