Definition of a variable

A variable can be considered as a box in which values can be placed.

Sometimes there is only one value that can be placed in the box and you have to determine what that value is.  At other times you get to pick a value that can be put into the box.  These two different situations will be explained below.

Do you remember when you were in primary school and you were learning about addition?  The teacher would hand you worksheets with instructions like the following:

Fill in the box:

  

\(\PalAb{\Box}+4=6\)

Variables can be used to represent the box.  Suppose we use an \(\PalAb{x}\) to replace the box.
Now we can say:

Find the value of \(\PalAb x\) for which 

  

\(\PalAb x+4=6\).

Why did we switch from boxes to letters?  We did this because letters are more practical.

For instance, the formula for finding the area of a square is:

\(\begin{align}\PalBc A&=\PalAa\ell \times \PalAa\ell\\&=\PalAa\ell^2\end{align}\)

The letter \(A\) represents the area and the letter \(\ell\) the length of a side of the square.
This formula makes more sense than, for example:

\(\PalBc\bigcirc=\PalAa\triangle^2\)

The two formulae say exactly the same thing, but using \(\PalBc A\) for area and \(\PalAa\ell\)  for length are more useful than using a circle and a triangle.  Boxes are fine, but letters are better.
