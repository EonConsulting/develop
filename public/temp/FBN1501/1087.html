The formula for calculating a composite index is:

  

For a composite price index, substitute the \(\PalBc n\) with \(\PalBc p\):

  
 

\(I_\PalAa n = \displaystyle\frac{\sum { \PalBc n_\PalAa n}}{\sum { \PalBc n_\PalAb0}} \times 100\)

  

\(I_\PalAa n = \displaystyle\frac{\sum { \PalBc p_\PalAa n}}{\sum { \PalBc p_\PalAb0}} \times 100\)

  

The sigma sign signifies a sum.  So, \(\sum \PalBc p_\PalAa n\) is the sum of the prices for the given period:

\( \begin{align} \sum \PalBc p_\PalAa n&=\sum { \PalBc p_{\PalAa{2009}}}\\&= 9\Ndp75+25\Ndp00+11\Ndp00+19\Ndp00+12\Ndp00+55\Ndp00\\ &=131\Ndp75 \\&=\PalAa{\Cur131\Cdp75}\end{align}\)

And \(\sum \PalBc p_\PalAb0\) is the sum of the prices for the base period: 

\( \begin{align} \sum \PalBc p_\PalAb0&=\sum { \PalBc p_{\PalAb{2000}}}\\&= 8\Ndp50+16\Ndp00+8\Ndp00+15\Ndp00+9\Ndp00+51\Ndp00\\ &=107\Ndp50\\&=\PalAb{\Cur107\Cdp50} \end{align}\)

Therefore, for this example the composite price index for food in 2009 with 2000 as the base year, is:

\(\begin{align}I_\PalAa n &=\displaystyle\frac{\sum { \PalBc p_\PalAa n}}{\sum { \PalBc p_\PalAb0}} \times 100\\&= \displaystyle\frac{\sum { \PalBc p_{\PalAa{2009}}}}{\sum {\PalBc p_{\PalAb{2000}}}} \times 100\\ &=\displaystyle\frac{\PalAa{\Cur131\Cdp75}}{\PalAb{\Cur107\Cdp50}} \times 100\\&= 122\Ndp56 \end{align}\)
