Variables in expressions

When working with variables, remember the following:

No matter how many times a variable is used in
an expression, the value is always the same.

Example with one variable used more than once

If you have one variable in an expression used more than once, then the variable represents the same value in each instance.

In the expression \(3k-2+8k\), the variable \(k\) is used twice. You cannot substitute two different numbers for \(k\). If \(\PalBa k=\PalBa6\), then \(6\) must be substituted for \(k\) in both places in the expression, as illustrated below:

Mathematical expression

  

\(3\PalBa k-2+8\PalBa k\)

Substitute \(\PalBa6\) for \(\PalBa k\)

\(=\)

\(3(\PalBa6)-2+8(\PalBa6)\)

Calculate

\(=\)

\((3\times6)-2+(8\times6)\)

Simplify

\(=\)

\(18-2+48\)
 

\(=\)

\(16+48\)
 

\(=\)

\(64\)

Example with two different variables

If you have two different variables in an expression, then most likely they will represent different values.

Evaluate the mathematical expression \(2r^2+1+s\) when \(\PalBb r=\PalBb4\) and \(\PalBc s=\PalBc7\). Substitute \(4\) for \(r\) and \(7\) for \(s\) in the expression, as illustrated below:

Mathematical expression

  

\(2\PalBb r^2+1+\PalBc s\)

Substitute \(\PalBb4\) for \(\PalBb r\) and \(\PalBc7\) for \(\PalBc s\)

\(=\)

\(2(\PalBb 4)^2+1+(\PalBc 7)\)

Calculate

\(=\)

\((2\times 16)+1+7\)

Simplify

\(=\)

\(32+1+7\)
 

\(=\)

\(33+7\)
 

\(=\)

\(40\)
