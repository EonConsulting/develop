Intervals

Sometimes we are given a chart showing frequencies of certain groups instead of the actual values.  We can still come up with a good estimate of a typical value for the set of data, provided that we make some assumptions.

We assume that the values in each interval
are spread evenly throughout the interval.

If this is the case, then the mean for each interval should be approximately equal to the midpoint for each interval.

The midpoint of an interval divides the interval into two equal parts
and is obtained by adding the upper and lower limits of each interval
and dividing the result by two.

This middle value represents the class interval in calculations.

Example

The tyre company Radial advertised that their XXX tyres will travel at least \(65\Tsp000\) kilometres before one of the four tyres will no longer meet the minimum safety requirements.  A frequency table has been drawn up, providing data about intervals.

The first interval in Radial's frequency table is \(13\Ndp5 - 24\Ndp5\).

The midpoint is found by adding the lower interval boundary to the higher interval boundary, then dividing that sum by \(2\):

\(\displaystyle\frac{\PalAa{13\Ndp5}+\PalAb{24\Ndp5}}{2}\\ =\displaystyle\frac{38}{2}\\ =\PalBc{19}\)

The midpoint for the interval \(\PalAa{13\Ndp5}-\PalAb{24\Ndp5}\) is \(\PalBc{19}\).
