The exponential function is an increasing function of \(x\).  As \(x\) increases from large negative values to large positive values, the function values constantly increase.

    

The exponential function is a decreasing function of \(x\).  As \(x\) increases from large negative values to large positive values, the function values constantly decrease.

  
 

In both cases the functions are positive for all values of \(x\) and they all pass through the point \((0;1)\).

Exponential functions with base \(\PalAb{a>1}\) are known as growth curves.

On the other hand, exponential functions with base \(\PalAa{0<a<1}\) are known as decay functions because of their decreasing property.  They are useful to model certain 'negative growth' phenomena such as radioactive decay processes, price-demand curves and non-linear depreciation.
