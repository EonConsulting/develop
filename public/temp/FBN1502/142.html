Function notation

It is useful to give a function a name. The most common name is \(f\), but you can have other names, like \(g\) or \(h\). We will use \(f\).

A function is written in a special way. The classic way of writing a function is \(f(x)=\dotso\).

Example

Add four is an example of a very simple function. This function is written as:

\( f(\PalAb {x})=\PalAb x+4\)

where \({f(\:)}\) represents the function name,
  \(\PalAb{x}\) represents the input
and \(\PalAb x+4\) represents the output, or what to do.

What goes into the function is put inside brackets \({(\:)}\) after the name of the function:

  \({f(}\PalAb{x}{)}\) shows you the function is called \( f\) and \(\PalAb x\) goes in.

Then what the function does with the input, namely the output, appears:

  \({f(}\PalAb{x}{)}=\PalAb x+4\) shows that the function \(f\) takes \(\PalAb x\) and adds \(4\).
