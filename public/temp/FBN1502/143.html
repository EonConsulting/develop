Examples

Consider the function \(f(\PalAb{x})=3\PalAb{x}\).  An input of \(\PalAb{5}\) gives an output of \(\PalAa{15}\).  Can you see why?

Function \(f(\PalAb{x})\) \(=\) \(3\PalAb{x}\)
Input \(\PalAb{5}\) \(\PalAb{x}\) \(=\) \(\PalAb{5}\)
Substitute \(\PalAb{x}\) in function \(f(\PalAb{5})\) \(=\) \(3(\PalAb{5})\)
Calculate and simplify   \(=\) \(3\times\PalAb{5}\)
Obtain output   \(=\) \(\PalAa{15}\)

Do not get too concerned about naming the input variable \({x}\).  It is just a placeholder and is there to show you where the input goes and what happens to it. It could be named anything.
