Add exponents

Look at a simple multiplication problem: \(\PalAa4\) \(\times\) \(\PalAb8\) \(=\) \({32}\)
Another way to write this is: \(\PalAa2\times\PalAa2\) \(\times\) \(\PalAb2\times\PalAb2\times\PalAb2\) \(=\) \({32}\)
Or it can be written as: \(2^\PalAa2\) \(\times\) \(2^\PalAb3\) \(=\) \({32}\)
Or as: \(2^\PalAa2\) \(\times\) \(2^\PalAb3\) \(=\) \(2^{\PalAa2+\PalAb3}\)

When we multiply powers that have the same base,
we can add the exponents to get the product.

\(a^\PalAa m\times a^\PalAb n = a^{(\PalAa m+\PalAb n)}\)

Example

\(\begin{align}2^\PalAa4\times2^\PalAb5&=2^{\PalAa4+\PalAb5}\\&=2^9\\&=512\end{align}\)
