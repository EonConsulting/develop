Identifying when not a function

A function may only result in one single
value for the dependent variable.

It might seem like all formulas could be considered functions.  However, this is not always the case, as can be seen in the following illustration:

Consider the formula

\(\PalAa y^2\)

\(=\)

\(\PalAb x\)

        

or

 \(y\) \(=\) \(\pm \sqrt{x}\)        

which is

 \(y\) \(=\) \( \sqrt{x}\) and \(y\) \(=\) \(- \sqrt{x}\)

When \(\PalAb x=\PalAb 4\), we get

 \(\PalAa y^2\) \(=\) \(\PalAb4\)        

or

 \(y\) \(=\) \(\pm \sqrt{4}\)        

which is

 \(y\) \(=\) \( \sqrt{4}\) and \(y\) \(=\) \(- \sqrt{4}\)

 

   \(=\) \( \sqrt{2\times2}\)     \(=\) \(- \sqrt{2\times2}\)
   

\(=\)

\(2\)

    

\(=\)

\(-2\)

So when \(\PalAb x=\PalAb 4\), we get

\(\PalAa y\)

\(=\)

\(\PalAa 2\)

and

\(\PalAa y\)

\(=\)

\(\PalAa{-2}\)

Look at the three parts of this formula \(\PalAa y^2=\PalAb x\) (or \(\PalAa y=\pm \sqrt{\PalAb x}\)) for different values of \(\PalAb x\):

Input
(independent variable)		 Relationship Output
(dependent variable)
\(\PalAb4\) \({\pm \sqrt{\PalAb{4}}}\) \(\PalAa2\) and \(\PalAa{-2}\)
\(\PalAb6\) \({\pm \sqrt{\PalAb{6}}}\) \(\PalAa{2\Ndp45}\) and \(\PalAa{-2\Ndp45}\)
\(\PalAb{16}\) \({\pm \sqrt{\PalAb{16}}}\) \(\PalAa4\) and \(\PalAa{-4}\)

Each time we input a value of \(x\), we get two different answers for \(y\). As we can see, when \(x\) equals \(4\), \(y\) equals \(2\) and \(-2\) and so forth.

Therefore, the formula \(\PalAa y^2=\PalAb x\) does not represent a function, because for each positive \(\PalAb x\) there are two values for \(\PalAa y\).
