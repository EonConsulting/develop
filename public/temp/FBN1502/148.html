Substitute values for variables

Function notation shows us how to substitute particular numerical values for each independent variable in a specific function.

Example

If we consider the function \(z=f(\PalAb x,\PalAa y)=6\PalAb x-3\PalAa y\), then \(f(\PalAb2;\PalAa3)\) means that \(\PalAb2\) is substituted for \(\PalAb x\) and \(\PalAa 3\) is substituted for \(\PalAa y\).  The value of \(f(\PalAb2;\PalAa3)\) is then calculated as follows:

If the function is \(z\) \(=\) \(f(\PalAb x,\PalAa y)\) \(=\) \(6\PalAb x-3\PalAa y\)
then \(f(\PalAb2;\PalAa3)\) is     \(f(\PalAb2;\PalAa3)\) \(=\) \(6(\PalAb2)-3(\PalAa3)\)
Calculate and simplify       \(=\) \(6\times\PalAb2-3\times\PalAa3\)
        \(=\) \(12-9\)
        \(=\) \(3\)

Similarly, the value of \(f(\PalAb{5\Ndp5};\PalAa{4\Ndp5})\) is calculated as follows:

If function is \(z\) \(=\) \(f(\PalAb x,\PalAa y)\) \(=\) \(6\PalAb x-3\PalAa y\)
then \(f(\PalAb{5\Ndp5};\PalAa{4\Ndp5})\) is     \(f(\PalAb{5\Ndp5};\PalAa{4\Ndp5})\) \(=\) \(6(\PalAb{5\Ndp5})-3(\PalAa{4\Ndp5})\)
Calculate and simplify       \(=\) \(6\times\PalAb{5\Ndp5}-3\times\PalAa{4\Ndp5}\)
        \(=\) \(33-13\Ndp5\)
        \(=\) \(19\Ndp5\)

Note that confusion may arise when using the decimal, so we rather use a semicolon \((;)\) instead of a comma \((,)\) to separate variables.
