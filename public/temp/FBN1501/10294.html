Example

Draw a box-and-whiskers diagram for the data below:

\(6\), \(3\), \(9\), \(8\), \(4\), \(10\), \(8\), \(4\), \(15\), \(8\), \(10\)

Solution

Order the data in ascending order.  There are \(\PalAa{11}\) values in the data set, so \(\PalAa{n}=\PalAa{11}\):

\(\PalAa{1^{\text{st}}}\) \(\PalAa{2^{\text{nd}}}\) \(\PalAa{3^{\text{rd}}}\) \(\PalAa{4^{\text{th}}}\) \(\PalAa{5^{\text{th}}}\) \(\PalAa{6^{\text{th}}}\) \(\PalAa{7^{\text{th}}}\) \(\PalAa{8^{\text{th}}}\) \(\PalAa{9^{\text{th}}}\) \(\PalAa{10^{\text{th}}}\) \(\PalAa{11^{\text{th}}}\)
\(3\) \(4\) \(4\) \(6\) \(8\) \(8\) \(8\) \(9\) \(10\) \(10\) \(15\)

Then, determine the \(Q_1\), \(Me\) and \(Q_3\):

For \({Q_1}\)

\(\PalAa{Q_1}\) is the value of the 

\(\PalAa{\frac{1}{4}(\PalAa n+1)^{\text{th}}}\\\)

\(=\frac{1}{4}(\PalAa {11}+1)\)

\(=\frac{12}{4}\)

\(=\PalAa{3^{\text{rd}}}\)

observation.
 

So, \(\PalAa{Q_1}\)

\(=\PalAa4\).

  

For \(Me\text{/}Q_2\)

\(\PalAb{Me\text{/}Q_2}\) is the value of the

\(\PalAb{\frac{1}{2}(\PalAa n+1)^{\text{th}}}\\\)

\(=\frac{1}{2}(\PalAa {11}+1)\)

\(=\frac{12}{2}\)

\(=\PalAb{6^{\text{th}}}\)

observation.
 

So, \(\PalAb{Me}\)

\(=\PalAb 8\).

  

For \(Q_3\)

\(\PalBc{Q_3}\) is the value of the

\(\PalBc{\frac{3}{4}(\PalAa n+1)^{\text{th}}}\\\)

\(=\frac{3}{4}(\PalAa {11}+1)\)

\(=\frac{36}{4}\)

\(=\PalBc{9^{\text{th}}}\)

observation.
 

So, \(\PalBc{Q_3}\)

\(=\PalBc{10}\).
