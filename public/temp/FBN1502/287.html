Cost as a function

A factory manufactures office desks.  The manufacturing costs for a month are given below:

  cost of wood and other materials amount to \(\Cur105\Cdp00\) per desk,
  direct labour costs \(\Cur175\Cdp00\) per desk,
  variable factory overheads amount to \(\Cur70\Cdp00\) per desk and
  fixed factory overheads amount to \(\Cur100\Tsp000\Cdp00\) for the month.

The total monthly cost \((\PalAa{TC})\) of the factory is calculated as

\( \begin{align} \PalAa{TC}&=\PalBc{\text{fixed cost}}+\PalAb{\text{variable cost}}\\ &=\PalBc{100\Tsp000}+\PalAb{350q}. \end{align} \)

The total monthly cost of the factory is a function of the number of desks manufactured \((\PalAb q)\) and can also be written as

\(\PalAa C(\PalAb q )=100\Tsp000+350\PalAb q.\)

A function can be written in many different ways.  Using \(\PalAb x\) instead of \(\PalAb q\) to represent the number of desks, the cost function can be written as

\(\PalAa C(\PalAb x )=100\Tsp000+350\PalAb x.\)

Instead of using \(\PalAa C\), we could also use \(\PalAa f\) and write the function as

\(\PalAa f(\PalAb x )=100\Tsp000+350\PalAb x\)

or use any other letter we want to.
