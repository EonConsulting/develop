Questions 7 and 8 are based on the following information:
Memory Lane Store sells old coins at a special price.  These coins cost the store \(\Cur20\Cdp00\) each, with a fixed cost of \(\Cur210\Cdp00\) per week for advertising.  The weekly cost, \(C(x)\), is a linear function of the number of coins bought, \(x\), and is given by

\(C(x)=210+20x\).

The revenue, \(R(x)\), from selling these coins (in rand) is given by the function

\(R(x)=-10x^2+400x\).

The profit function, \(P(x)\), is given by

\(P(x)=R(x)-C(x)\).

Question 7

The marginal profit function is:

  1. \(P'(x)=-10x^2+380x-210\)
  2. \(P'(x)=-20x+380\)
  3. \(P'(x)=-20x+400\)
  4. \(P'(x)=-20x+420\)
  5. None of the above

Question 8

The number of coins that has to be sold in order to maximise the weekly profit is:

  1. \(370\)
  2. \(3\Tsp400\)
  3. \(19\)
  4. \(105\)
  5. None of the above
