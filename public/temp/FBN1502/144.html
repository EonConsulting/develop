Identifying a function

Input, relationship and output

Does a function always have a name, such as \( f(x)\) or \(k(p)\)?  Sometimes a function has no name, and you might just see something like:

\(\PalAa y=3\PalAb x +2\)

However, we still have:

an input \(\PalAb x\)
a relationship multiplying by \(3\) and adding \(2\)
and an output \(\PalAa y\).

Single value result

There is another important concept about a function we need to consider, namely a function may only result in a single value for the dependent variable.

So, does the formula \(\PalAa y=3\PalAb x +2\) represent a function?  Yes it does, because every time that you substitute a number for \(\PalAb x\), you will only get a single value for \(\PalAa y\).

To illustrate, we can substitute different values for \(\PalAb x\) and get different single values for \(\PalAa y\):

Formula \(\PalAa y\) \(=\) \(3\PalAb x+2\)
Let \(\PalAb x = \PalAb 3\) \(\PalAa y\) \(=\) \(3(\PalAb 3)+2\)
    \(=\) \(3\times\PalAb 3+2=9+2=\PalAa{11}\)
When \(\PalAb x = \PalAb 3\), then \(\PalAa y\) \(=\) \(\PalAa{11}\)
Let \(\PalAb x = \PalAb 7\) \(\PalAa y\) \(=\) \(3(\PalAb 7)+2\)
      \(3\times\PalAb 7+2=21+2=\PalAa{23}\)
When \(\PalAb x = \PalAb 7\), then \(\PalAa y\) \(=\) \(\PalAa{23}\)

So, we see that when \(\PalAb x\) equals \(\PalAb 3\), \(\PalAa y\) equals \(\PalAa{11}\) every time.  No other number will correspond with \(\PalAb 3\), when using this formula.

And if we input another number, like \(\PalAb 7\), we get a different output.  When \(\PalAb x\) equals \(\PalAb 7\), \(\PalAa y\) equals \(\PalAa{23}\) every time.  No other number will correspond with \(\PalAb 7\), when using this formula.

Therefore, this formula can be labelled a function.

This seems straightforward.  It would seem like all formulas could be considered functions.  However, this is not always the case.
