Formulas

It is possible to calculate the mean, variance and standard deviation where we have not been provided with the raw data, but data grouped into intervals, with frequencies associated with each interval.

The assumption we make is that each interval is represented by its midpoint \(\PalBc x\).

The standard deviation is the positive square root of the variance:

Standard deviation is 

\(S=\sqrt{S^2}\)
 

The formulas for finding the variance for both types of data are:

Raw data variance is  

\(S^2=\displaystyle\frac{\sum_{i=1}^\PalAa n (\PalBc x_i-\overline{\PalBc x})^2}{\PalAa n-1}\) 

where \(\PalAa n\) is the number of observations in the data set.

Data in frequency distribution variance is

\(S^2=\displaystyle\frac{\sum_{i=1}^\PalAa n (\PalBc x_i-\overline{\PalBc x})^2}{\PalAa n-1}\) 

where \(\PalAa n=\sum_{i=1}^\PalAb kf_i\) and \(\PalAb k\) is the number of intervals.
