Consider the exponential function 

\(f(x)=2^x\).

To evaluate this function, choose a numerical value for \(x\) and calculate the function's value at this \(x\).

If \(x\) has positive values, for example if \(x=\PalAa 3\), then

\(\begin{align} f(\PalAa3)&=2^\PalAa3\\ &=2\times2\times2\\ &=\PalAa8 \end{align}\)

If \(x\) has negative values, for example if \(x=\PalAb{-3}\), then

\(\begin{align} f(\PalAb{-3})&=2^\PalAb{-3}\\ &=\displaystyle\frac{1}{2^3}\\ &=\PalAb{\displaystyle\frac{1}{8}} \end{align}\)
