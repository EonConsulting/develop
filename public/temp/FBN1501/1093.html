Formulas

We can calculate composite indexes for the following four indexes:

 

Base-weighted price index

The Laspeyres price index

\(P_L\left(\PalAa n \right) =\displaystyle \frac{\sum{\PalBc p_\PalAa n \PalBc q_\PalAb0}}{\sum{ \PalBc p_\PalAb0\PalBc q_\PalAb0 }}\times 100\)

How much will a basket of goods consumers bought in the base period, cost in the current period?

    

Base-weighted quantity index

The Laspeyres quantity index

\(Q_L\left(\PalAa n \right) =\displaystyle \frac{\sum{\PalBc p_\PalAb 0 \PalBc q_\PalAa n}}{\sum{ \PalBc p_\PalAb0\PalBc q_\PalAb0 }}\times 100\)

  
 

Current-weighted price index

The Paasche price index

\(P_P\left(\PalAa n \right) =\displaystyle \frac{\sum{\PalBc p_\PalAa n \PalBc q_\PalAa n}}{\sum{ \PalBc p_\PalAb0\PalBc q_\PalAa n }}\times 100\)

How much would a basket of goods consumers buy in the current period, have cost in the base period?

    

Current-weighted quantity index

The Paasche quantity index

\(Q_P\left(\PalAa n \right) =\displaystyle \frac{\sum{\PalBc p_\PalAa n \PalBc q_\PalAa n}}{\sum{ \PalBc p_\PalAa n\PalBc q_\PalAb0 }}\times 100\)

  
 

In all cases:

\(\PalBc p_\PalAa n\) is the price in the \(\PalAa {n^{\text{th}}}\) year;
\(\)\(\PalBc p_\PalAb0\) is the price in the base year;
\(\PalBc q_\PalAa n\) is the quantity in the \(\PalAa {n^{\text{th}}}\) year;
\(\PalBc q_\PalAb 0\) is the quantity in the base year.
