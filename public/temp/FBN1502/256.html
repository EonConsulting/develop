Defining a system of linear equations in two variables

What is a system of linear equations in two variables?

It is a set or collection of linear equations whose common solutions are looked for.  The simplest linear system is one with two equations and two variables.  

We can also refer to this kind of system as simultaneous linear equations or a linear system.

Consider the following system with two linear equations and two variables:

\(\PalAa{y=\displaystyle\frac{5}{2}x-5}\) and \(\PalAb{y=-x+1}\).

Since the two equations are in a system, we work with them at the same time.  In particular, we can graph them together on the same set of axes:

A solution for a single equation is any point that lies on the line for that equation.  A solution for a system of equations is any point that lies on each line in the system.  

For example, note the following three points on the graph: 

  • The turquoise point is not a solution of the system, because it is not on either line.
  • The purple point is also not a solution of the system, because it lies on only one of the lines, not on both of them.
  • The green point is a solution of the system, because it lies on both of the lines.  In particular, this point is the intersection of the two lines.  Since this point is on both lines, it satisfies (or is a solution of) both equations.  So it satisfies the entire system of equations.

For systems of equations, the solutions are the points of intersections of the corresponding lines.

You can confirm the solution by plugging it into the system of equations and confirming that the solution satisfies each equation.
