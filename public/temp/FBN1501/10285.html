Definition

The mode \((Mo)\) of a data set is that value which occurs most often.

Example

A survey of \(14\) employees' performance scores out of \(50\) had these results, ordered from lowest to highest:

\(33\) \(33\) \(35\) \(36\) \(37\) \(38\) \(38\) \(\PalAb{39}\) \(\PalAb{39}\) \(\PalAb{39}\) \(40\) \(40\) \(42\) \(45\)

Let us look at the data in a different way:

The mode of the data is \(\PalAb{39}\) because \(39\) occurs three times, which is more than any other score.
