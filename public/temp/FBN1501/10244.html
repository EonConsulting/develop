Remarks about roots

The properties of square roots also apply to roots with higher indexes.
Please take note of and remember the following about roots:

  • Another way to write roots is to write them as fractional exponents, for example:

\(\begin{align} \sqrt{16}&=16^{\PalAb{\frac{1}{2}}}\\&=4 \end{align} \)

\(\begin{align} \sqrt[\PalAb3]{27}&=27^{\PalAb{\frac{1}{3}}}\\&=3 \end{align} \)

\(\begin{align} \sqrt[\PalAb4]{16}&=16^{\PalAb{\frac{1}{4}}}\\&=2 \end{align} \)

\(\begin{align} \sqrt[\PalAb n]{x}&=x^{\PalAb{\frac{1}{n}}}\\ \end{align} \)

 

  • Note that roots can be added and subtracted only if the same number is under the root sign, for example:

\(6\sqrt{\PalAa3}+4\sqrt{\PalAa3}=10\sqrt{\PalAa3}\)

But the following roots cannot be added and simplified:

\(6\sqrt{\PalAb2}+4\sqrt{\PalAa3}\)

 

  • Be careful when simplifying \(\sqrt{25-16}\) :

\(\begin{align}\sqrt{\PalAa{25-16}}&=\sqrt{\PalAa9}\\ &=3\end{align}\)

Do not make the following mistake:

\(\begin{align}\sqrt{25-16}&=\sqrt{25}-\sqrt{16}\\ &=5-4\\ &=1 \end{align}\)

Remember: Finish all the calculations inside the square root first before determining the square root.

 

  • The property \(\sqrt{\PalAa a\PalAb b}=\sqrt{\PalAa a}\times\sqrt{\PalAb b}\) only applies to numbers in roots that are multiplied.

Remember that \(\sqrt{a+b}\PalAb\ne \sqrt{a}+\sqrt{b}\) and \(\sqrt{a-b}\PalAb\ne \sqrt{a}-\sqrt{b}\).
