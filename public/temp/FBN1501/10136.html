Order of operations

Evaluating mathematical expressions can be a simple process, but needs to follow an order of operations to get the right answer.  The following sequence describes the order you follow to add, subtract, multiply and divide.  The order is: 

B. E. M. D. A. S

Brackets (B)

Perform the operations inside the brackets first.

Exponents (E)

Then exponents.

Multiplication (M)

Then multiplication, from left to right.  

If more than one operation have the same priority, operate from left to right.

\(\times\) and \(\div\) have the same priority.

Division (D)

Then division, from left to right.

If more than one operation have the same priority, operate from left to right.

\(\times\) and \(\div\) have the same priority.

Addition (A)

Then addition, from left to right.

If more than one operation have the same priority, operate from left to right.

\(+\) and \(-\) have the same priority.

Subtraction (S)

Then subtraction, from left to right.

If more than one operation have the same priority, operate from left to right.

\(+\) and \(-\) have the same priority.

This also represents the order of operations.
