The inverse function

We need to understand the term inverse function before we look at logarithmic functions.

Consider the function \(\PalAa{f(x)}\):

   

\(y=\PalAa{f(x)}=x+2\)

    
  \(x\)   \(x+2\)   \(y\)  
  \(\PalAa3\) \(\to\) \(3+2\) \(\to\) \(\PalAb5\)  
  \(\PalAa x\) goes in       \(\PalAb y\) comes out  

 

The inverse function just goes the opposite way.  The inverse function is written as \(\PalAb{f^{-1}(y)}\):

   

\(x=\PalAb{f^{-1}(y)}=y-2\)

    
  \(y\)   \(y-2\)   \(x\)  
  \(\PalAb5\) \(\to\) \(5-2\) \(\to\) \(\PalAa3\)  
  \(\PalAb y\) goes in       \(\PalAa x\) comes out  

 

The inverse function gives back the original value.

If the function \(\PalAa f\) turns the lecturer into a frog, then the
inverse function \(\PalAb{f^-1}\) turns the frog back into the lecturer.
