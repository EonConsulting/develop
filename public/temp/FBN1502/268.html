Example

Solve the inequality \(5x-15<0\).

Inequality in one variable

\(5\PalAb x-15\)

\(<\)

\(0\)

Add \(\PalAa{+15}\) both sides

 \(5\PalAb x-15\PalAa{+15}\) \(<\) \(0\PalAa{+15}\)
 

\(5\PalAb x\)

\(<\)

\(15\)

Divide both sides by \(\PalAa5\) 

\(\displaystyle\frac{5\PalAb x}{\PalAa5}\)

\(<\)

\(\displaystyle\frac{15}{\PalAa5}\)

 

\(\PalAb x\)

\(<\)

\(3\)

This is indicated on the number line by highlighting all the points for which the inequality is true.  The arrow shows that all the values on the number line less than \(3\) are in the solution.  Since the point \(x=3\) is not included, it is shown by means of an open circle.

\(\PalAb{x<3}\)

If the solution had been \(\PalAa x\) is less than or equal to \(\PalAa3\), the circle would be a dark, or filled in, circle.

\(\PalAa{x\leq3}\)

How can we check our answer?

We cannot use \(3\) to substitute in the inequality, because it lies outside our solution.  To check, we can choose any value that lies in the solution and substitute it into the inequality. Let us use \(\PalAb{-2}\).

Inequality in one variable

\(5\PalAb x-15\)

\(<\)

\(0\)

Substitute \(\PalAb{x=-2}\) in

\(5( \PalAb {-2})-15\)

\(<\)

\(0\)

 

\(5\times\PalAb{-2}-15\)

\(<\)

\(0\)
 

\(-10-15\)

\(<\)

\(0\)
Result is true

\(-25\)

\(<\)

\(0\)

Our substitution gave a true result, so the solution is correct.
