Graphing functions

What does the graph of the logarithmic function look like? 

Consider the exponential function 

\(f(x)=2^x\).

In this case the base is equal to \(2\).  The logarithmic function is the inverse of the exponential function.  Remember that the inverse function should give you back the original value that was put into the function.  Therefore, if \(\PalAa{(4;16)}\) is a point on the graph of an exponential function, then \(\PalAb{(16;4)}\) would be the corresponding point on the graph of the logarithmic function.

Look at the following explanation, using this exponential function:

Exponential function Logarithmic function
\(x\) \(\to\) \(f\) \(\to\) \(y\) \(\to\) \(f^{-1}\) \(\to\) \(x\)
\(-1\) \(\to\) \(2^{-1}\) \(\to\) \(0\Ndp5\) \(\to\) \(\log_2{0 \Ndp5}\) \(\to\) \(-1\)
\(\PalAa{-1}\) goes in and \(\PalAa{0\Ndp5}\) comes out \(\PalAb{0\Ndp5}\) goes in and \(\PalAb{-1}\) comes out
The point is \(\PalAa{(-1;0\Ndp5)}\). The point is \(\PalAb{(0\Ndp5;-1)}\).
