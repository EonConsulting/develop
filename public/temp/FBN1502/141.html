The concept of a function

When the value of one variable is dependent on the value of another variable (or several other variables), we make the statement that the dependent variable is a function of the independent variable.

Value

The independent variable determines the value of the dependent variable:

  • The way in which the value of the dependent variable is determined must be clearly stated and be unambiguous.
  • Only one single value for the dependent variable must result.

Relationship

A function is a relationship between two variables:

  • The independent variable determines the value of the dependent variable.
  • A function relates an input (independent variable) to an output (dependent variable).
  • This may result in only one value for the second (dependent) variable.

Parts of a function

A function has three main parts:

  • Input
  • Relationship
  • Output

Example
