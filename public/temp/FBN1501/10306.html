Equal parts

The median \((Me)\)is that value which separates a sorted data set into two equal parts:

If we divide a sorted data set into four equal sized parts we get quartiles:

\(\PalAa{Q_1}\)

The first quartile represents the value indicating the end of the first \(\PalAa{25\%}\) of the data values.

\(\PalAb{Q_2}\)

The second quartile represents the value indicating the end of the second \(\PalAb{25\%}\) of the data values (or the value which divides the data set into two equal parts).  The second quartile has the same value as the median.

\(\PalBc{Q_3}\)

The third quartile represents the value indicating the end of the third \(\PalBc{25\%}\) of the data values.
